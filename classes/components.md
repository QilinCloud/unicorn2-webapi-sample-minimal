# ApiWeb Sample Classes Components

- `class.config.php`: reads `config.php`.
- `class.logger.php`: writes level-controlled logs.
- `class.security.php`: validates and emits HMAC headers.
- `class.request.php`: parses the JSON envelope.
- `class.answer.php`: serializes and signs responses.
- `class.result.php`: stores per-object results.
- `class.error.php`: stores protocol and item errors.
- `interface/`: Unicorn DTO and enum mirrors.
